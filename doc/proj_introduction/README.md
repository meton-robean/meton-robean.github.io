## 项目仓库介绍

### [Func-Loop-Profiler](https://github.com/meton-robean/Func-Loop-Profiler)  
基于PIn动态插桩框架的函数-循环嵌套关系分析工具,可以分析程序执行的函数-循环的嵌套关系、找出程序中耗时并且有优化潜力的循环、提供类似gprof的终端显示和可视化流图显示.
